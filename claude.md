# Claude Project Notes

## Package Manager

This project uses **pnpm**, not npm or yarn.

Use pnpm commands:

- `pnpm install` - Install dependencies
- `pnpm add <package>` - Add a package
- `pnpm update <package>` - Update a package
- `pnpm run <script>` - Run a script
